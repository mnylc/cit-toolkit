## Installation

Here are the steps to install:

0. `$ git clone git@github.com:mnylc/metro.org.git && cd metro.org`
0. `$ bundle install`
0. `$ bundle exec jekyll serve --watch`

The site should now be available at [http://127.0.0.1:4000](http://127.0.0.1:4000)
